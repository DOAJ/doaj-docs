Article XML Upload: DOAJ XML

Download Testset

1. Try uploading without providing a file

Test context
  • role: publisher
ID Action Expected Result  
M.1.1 Go to the "Upload Article XML" tab in the "Publisher Area" /publisher/uploadfile    
M.1.2 Do not fill in any inputs    
M.1.3 Click "Upload"    
M.1.3.1   Under the Select a file input and Upload from a link input an error "You must specify the file or upload from a link" error is displayed

2. Upload a file which is not XML

Test context
  • role: publisher
ID Action Expected Result  
M.2.1 Go to the "Upload Article XML" tab in the "Publisher Area" /publisher/uploadfile    
M.2.2 Select "Choose file" and select a file from your computer which is not text based (e.g. an image file or a PDF)    
M.2.3 Click "Upload"    
M.2.3.1   A flash message with appropriate text appears at the top of the screen, indicating an error has occurred
M.2.3.2   Your file is shown in the "History of uploads" with the status "processing failed" and a suitable entry in the "Notes". Check that the explanation link goes to a suitable reason and resolution for the problem.
