1. Reject, Unreject and Accept

Test context
  • role: admin
Action Expected Result
Create an Update Request for a journal that does not have a current Update Request. Make a change that will be fairly easy to review from the ToC (e.g. the APC amount). Take a note of the Journal's ToC URL
Locate the Update Request created in the previous step from the admin area search
  • The record contains the value "Update Request For: [journal id]"
Click "Review Update"
Change the Application Status to "Rejected
Click "Save"
Click the "Unlock and Close" button / close the window
Go to the "My update requests" page of the publisher dashboard
  • The update request is no longer visible
Find the Update Request again via the search (e.g. filter by Rejected and sort by last updated most recent first)
  • Update Request says "Originally For: [journal id]"
Click "View Finished Update"
Change the Application Status to "Pending"
Click "Save"
Click the "Unlock and Close" button / close the window
Go to the "My update requests" page of the publisher dashboard
  • The update request is visible again in the status "Pending"
Find the Update Request again via the search (e.g. filter by Pending and sort by last updated most recent first)
  • The record contains the value "Update Request For: [journal id]"
Click "Review Update"
Change the Application Status to "Accepted"
Click "Save"
Click the "Unlock and Close" button / close the window
Go to the Journal ToC
  • The Journal page has been updated with the new information from the Update Request (e.g the updated APC amount)
Search for the Journal's ISSN in the admin journal search
  • There is only one Journal published in DOAJ with that ISSN
Find the accepted application via the admin search (e.g. filter by Accepted and sort by last updated most recent first)
  • Update Request says "Produced Journal: [journal id]"
