1. Test Editors Journal Search

Test context
  • role: Editor
ID Action Expected Result  
V.1.1 Go to the DOAJ editor's journal search at /editor/group_journals    
V.1.1.1   All available records are found, and the first page of results is shown
V.1.2 Open and Collapse all the facets on the left    
V.1.2.1   The facet opens and closes as expected
V.1.2.2   Each facet contains a maximum of 10 values relevant to that field
V.1.3 Open one of the facets of your choice    
V.1.4 Select one of the values for that facet    
V.1.4.1   All the results for the selected constraints are found, and the first page is shown
V.1.4.2   The filter applied is shown in the facets on the left and above the search results
V.1.5 Go through each sort option in the "sort by" pull-down    
V.1.6 In each case, click the "sort by" button to reverse the direction of sort    
V.1.6.1   The current search results are ordered correctly according to the selected sort field and the selected sort direction
V.1.7 Enter a search term in the search box    
V.1.7.1   All the results for the selected constraints are found, and the first page is shown
V.1.7.2   Your search text ramains in the search box
V.1.8 Go through each field constraint under the "search all" pull-down. You may change your search text if no results are found, to demonstrate that each field works correctly    
V.1.8.1   If there are search results, the first page is displayed
V.1.8.2   If there are no search results, suitable "no results match" text is displayed
V.1.9 Change the page size, using the page-size pull-down    
V.1.9.1   The size of the page of results displays increases appropriately
V.1.10 Click the "Next" button to go to the next page of results    
V.1.10.1   The next page of results is displayed
V.1.10.2   The "First" and "Prev" buttons are activated
V.1.11 Click "Next" again    
V.1.12 Click "Prev"    
V.1.12.1   You are navigated one page forward and then one page back in the results
V.1.13 Click "First"    
V.1.13.1   You are taken to the first page of the results
V.1.14 Click the black X button to clear all filters    
V.1.14.1   The interface returns to the initial state as per 1.1.1
V.1.15 Click "Edit this journal" link at the bottom of any journal    
V.1.15.1   /editor/journal/<id> opens in a new tab
