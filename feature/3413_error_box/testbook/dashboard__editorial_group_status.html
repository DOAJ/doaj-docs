1. Editorial Group Status for ManEds - Record Counts

Test context
  • role: admin
Action Expected Result
To prepare to do this test set up at least 2 editorial groups with multiple members and a good selection of applications assigned to a number of associate editors. You should set yourself as the managing editor of those groups
Go to your ManEd Dashboard page (/dashboard)
Scroll to the bottom of the dashbaord page, after your TODO list items
  • The editorial groups you manage are listed under the heading Activity
  • The first group is already selected and the data shown
Click on one of the other group names
  • The editorial group's statistics are presented on the screen
  • You can see the total number of applications assigned to the group
  • You can see the number of applications assigned to each editor
  • You can see how many applications have not been assigned
  • You can see the breakdown of the statuses of all the applications
Click on the application count next to the group's name (you may want to right click to keep the dashboard tab open during this test)
  • You are taken to the application search which shows the open applications assigned to this group.
  • The number of search results is the same as shown on the dashboard
Go back to the dashboard page
Click on an editor's name
  • A mail window opens in your mail client
Click on the application count next to an editor's name
  • You are taken to the application search which shows the open applications assigned to the user
  • The number of search results is the same as shown on the dashboard
Go back to the dashboard page
Click on the application count next to the "unassigned" label
  • You are taken to the application search which shows the open applications with no assigned editor
  • The number of search results is the same as shown on the dashboard
Go back to the dashboard page
Click on one of the statuses under "Applications By Status"
  • You are taken to the application search which shows the open applications in that status
  • The number of search results is the same as shown on the dashboard

2. Editorial Group Status for ManEds - Updating Editors

Test context
  • role: admin
Action Expected Result
To prepare to do this test set up at least 2 editorial groups with multiple members and a good selection of applications assigned to a number of associate editors. You should set yourself as the managing editor of those groups
Go to your ManEd Dashboard page (/dashboard)
Select an editorial group to see its status information
Take a note of the number of applications assigned to one of your editors
Click on the count of applications next to the "unassigned" tag
Click "Review Application" on the application in the search interface
Assign the application to the editor selected above
Go back to the dashboard page (you may need to refresh it if you kept the page open)
  • The editor selected now has one more application assigned to them
  • The number of unassigned applications has reduced by one
Click on the count of applications next to the selected editor
  • The application search is shown
  • The application you assigned to the editor is listed in the search

3. Editorial Group Status for ManEds - Changing Status

Test context
  • role: admin
Action Expected Result
To prepare to do this test set up at least 2 editorial groups with multiple members and a good selection of applications assigned to a number of associate editors. You should set yourself as the managing editor of those groups
Go to your ManEd Dashboard page (/dashboard)
Select an editorial group to see its status information
Take a note of the number of applications assigned to a specific status (e.g. in progress)
Click on the link to applications in that status
Click "Review Application" on the application in the search interface
Change the status to something different
Go back to the dashboard page (you may need to refresh it if you kept the page open)
  • The status you selected now has one more application in that state
  • The previous status has one fewer application in that state
Click on the new status link
  • The application search is shown
  • The application you put into this status is visible
