  • If there are search results, the first page is displayed

2. Test Public Search Results Display: Journals

Test context
  • role: anonymous
Action Expected Result
Go to the DOAJ search page at /search/journals
Under 'See journals' facet on the left, click 'With a DOAJ Seal'
  • All Journals have the DOAJ Seal displayed in the search results
Clear the "Journal has DOAJ Seal" filter
  • All Journals are shown
Open the facet "Journal License"
Select any CC licence from this facet
  • All Journals have the appropriate CC licence displayed in the search results
Clear the "Journal License" CC filter
Select "Publisher's own licence" from that facet
  • All Journals have the text "Publisher's own license" displayed in the search results
Clear the "Journal License" filter
Check the 'Without APCs or other fees' facet
  • All Journals have 'No charges' displayed in the search results
Clear the "Article processing charges (APCs)" facet
Find a journals with APCs (Sort on Added to DOAJ (oldest first) to see some)
  • All journals have 'APCS: amount + currency' displayed in the search results
Look at the first Journal result in your current search results
Click on its title
  • You are taken to the DOAJ ToC page for this journal
Click back in your browser window so you are back to the search results
Click on the word 'Website'
  • You are taken to the Journal's home page which opens in a new tab

Test context
  • role: anonymous
Action Expected Result
Go to the DOAJ search page at /search/articles
  • Only articles are shown in the results
