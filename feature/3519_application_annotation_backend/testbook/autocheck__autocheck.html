1. Preparation

Test context
  • role: Any Authenticated User
ID Action Expected Result  
Q.1.1 Activate the testdrive setup for this test suite, by using the following url

Application Link: http://testdoaj.cottagelabs.com//testdrive/autocheck 		   
Q.1.1.1   You receive the testdrive setup details, including an admin account for you to use, and an application and journal which have been autochecked for testing
Q.1.2 Log into DOAJ using the admin credentials supplied by the testdrive. You may want to do this in a private browsing window, so you can remain logged in to your own account in your main window.

Application Link: http://testdoaj.cottagelabs.com//account/login 		   
Q.1.2.1   You are logged in as an admin user
Q.1.3 You may now proceed to the tests below    

2. Application autochecks

Depends On
  • Autocheck : Autocheck : Preparation
Test context
  • role: Admin
ID Action Expected Result  
Q.2.1 Go to the Application admin_url supplied by the testdrive setup    
Q.2.1.1   You are looking at a test Application created for this test
Q.2.2 Scroll down to find the ISSN (print) and ISSN (online) fields    
Q.2.2.1   Both fields are annotated with a green tick, and text which says that ISSN is fully registered at ISSN.org
Q.2.3 Click on one of the "see record" links in the annotation    
Q.2.3.1   The link is opened in a new window/tab
Q.2.3.2   You are taken to the ISSN.org record. Note that for the purposes of this test, this is a random record on ISSN.org, unrelated to the actual record in DOAJ.
Q.2.4 close the ISSN.org window/tab and return to the application form    
Q.2.5 Click the "Dismiss" button on both annotations by the ISSN fields    
Q.2.5.1   The annotations are removed from the form, and replaced with text indicating that the check was dismissed
Q.2.5.2   On the right-hand side of the form, beneath the notes, there is a section "Dismissed Autochecks" which lists both annotations you just dismissed. This section is beneath the scrolling notes section, you may need to hover and scroll the notes section to see it.
Q.2.6 Click "Undismiss" by the text on the form for the ISSN (print) field which indicates that a check was dismissed    
Q.2.6.1   The annotation is restored to the form
Q.2.6.2   The annotation disappears from the "Dismissed Autochecks" section, leaving only one record there now
Q.2.7 Click "Undismiss" by the record in the "Dismissed Autochecks" section    
Q.2.7.1   The annotation is restored to the form
Q.2.7.2   The annotation disappears from the "Dismissed Autochecks" section, leaving no records there now
Q.2.8 Scroll to the "Best Practice" section of the application form, and look at the "Long-term preservation services" question    
Q.2.8.1   2 checks are visible attached to this question, one for CLOCKSS and one for LOCKSS, indicating that they are present in Keepers Registry
Q.2.9 Click on one of the "see record" links in the annotation    
Q.2.9.1   The link is opened in a new window/tab
Q.2.9.2   You are taken to the ISSN.org record. Note that for the purposes of this test, this is a random record on ISSN.org, unrelated to the actual record in DOAJ.
Q.2.10 close the ISSN.org window/tab and return to the application form    
Q.2.11 Click "Dismiss" by both notifications    
Q.2.11.1   The annotations are removed from the form, and replaced with text indicating that the check was dismissed
Q.2.11.2   The "Dismissed Autochecks" section lists both checks you just dismissed
