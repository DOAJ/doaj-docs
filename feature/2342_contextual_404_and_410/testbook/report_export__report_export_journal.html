2. Date range histograms

Test context
  • role: Admin
ID Action Expected Result  
BX.2.1 Go to the journal search page in the admin area

Application Link: http://testdoaj.cottagelabs.com//admin/ 		   
BX.2.1.1   At the bottom of the list of facets is a "Reporting Tools" section, including 3 options for working with dates "limit by date range", "created date histogram" and "last update histogram"
BX.2.2 Click on "Created Date Histogram"    
BX.2.2.1   A histogram of the number of journals created per day for the entire life of the DOAJ is displayed, grouped by year
BX.2.3 Click on one of the years    
BX.2.3.1   The search updates, and only records created in that year are displayed
BX.2.3.2   The search filter list at the top of the results indicates the created date range (it may indicate the start and end months of the year)
BX.2.4 Go back to the Created Date Histogram facet, and change the "Granularity" from "Year" to "Month"    
BX.2.4.1   The histogram updates to show the number of journals created per month for the selected year
BX.2.5 Click on one of the months    
BX.2.5.1   The search updates, and only records created in that month are displayed
BX.2.5.2   The search filter list at the top of the results indicates the selected month
BX.2.6 Go back to the Created Date Histogram facet. Above it is the "Limit by Date Range" facet, open this by clicking on it    
BX.2.6.1   The "Limit by Date Range" facet opens, showing a date range picker
BX.2.6.2   Beneath the date range picker the "Created Date" month you selected is also shown
BX.2.7 Remove the Created Date limit by clicking the "X" either in the "Limit by Date Range" section or at the top of the search results    
BX.2.7.1   The search results update to show all records
BX.2.8 Repeat the above steps for the "Last Update Histogram"    

3. Date range limiting

Test context
  • role: Admin
ID Action Expected Result  
BX.3.1 Go to the journal search page in the admin area

Application Link: http://testdoaj.cottagelabs.com//admin/ 		   
BX.3.1.1   At the bottom of the list of facets is a "Reporting Tools" section, including 3 options for working with dates "limit by date range", "created date histogram" and "last update histogram"
BX.3.2 Click on "Limit by Date Range"    
BX.3.2.1   A date range picker is displayed
BX.3.3 Open the "Type" pull-down menu    
BX.3.3.1   The options are "Created Date" and "Last Updated"
BX.3.4 If not already selected, select "Created Date"    
BX.3.4.1   The From and To dates show the month and year of the earliest and latest created dates in the search results, respectively
BX.3.5 Change the "Type" to "Last Updated"    
BX.3.5.1   The From and To dates show the month and year of the earliest and latest last updated dates in the search results, respectively
BX.3.5.2   Note that January 1970 is the date used for records with no last updated date
BX.3.6 Select a new Year value for the Last Updated date range and click "Apply"    
BX.3.6.1   The search results update to show only records that have been updated in the selected range
BX.3.6.2   The search filter list at the top of the results indicates the last updated date range, including the month and year
BX.3.6.3   The "Limit by Date Range" facet also shows the search filter for the Last Updated date range
BX.3.7 Try changing the month and year in both From and To fields for Last Updated in the same way    
BX.3.7.1   The search results and displayed filters reflect the new date ranges
BX.3.8 Choose a "To" date that is earlier than the "From" date    
BX.3.8.1   The "Apply" button is disabled
BX.3.9 Change the "Type" back to "Created Date"    
BX.3.10 Select a new range for the Created date range and click "Apply"    
BX.3.10.1   The "Last Updated" date filter is still active
BX.3.10.2   The search results update to show only records that have been created in the selected range (and were updated in the selected range)
BX.3.10.3   The search filter list at the top of the results indicates the created date range and the last updated range, including the month and year
BX.3.10.4   The "Limit by Date Range" facet also shows the search filter for the Created date range and the last updated range
BX.3.11 Remove both the Created Date and Last Updated date ranges by clicking the "X" in the "Limit by Date Range" section    
BX.3.11.1   The search results update to show all records

4. Export Search as CSV

Test context
  • role: Admin
ID Action Expected Result  
BX.4.1 Go to the journal search page in the admin area

Application Link: http://testdoaj.cottagelabs.com//admin/ 		   
BX.4.1.1   At the bottom of the list of facets is a "Reporting Tools" section, including 3 options for working with dates "limit by date range", "created date histogram" and "last update histogram"
BX.4.2 Open the "Limit by Date Range" facet and set a date range which gives you a subset of the data    
BX.4.3 Apply any other filters from the facets to further refine the search results    
BX.4.4 Click on "Export Data as CSV"    
BX.4.4.1   An export section expands which includes a section to export the search results
BX.4.5 Enter a name for the export in the input box and click "Generate"    
BX.4.5.1   a pop-up appears with a message that the export is being generated
BX.4.6 Close the popup    
BX.4.7 Go to the background jobs page in the admin area

Application Link: http://testdoaj.cottagelabs.com//admin/background_jobs 		   
BX.4.7.1   The export job is listed with the name "admin_reports by [your username]" (it may have a status of queued or processing, or even complete)
BX.4.8 Wait for the job to complete (this can take up to 10 minutes). You will need to refresh the page until it is in the "complete" status.    
BX.4.9 Open your "Notifications" menu at the top of the page    
BX.4.9.1   A notification appears that the report is ready for download
BX.4.9.2   A second notification saying a background job is finished will also be present
BX.4.10 Click on the notification that the report is ready for download    
BX.4.10.1   A download box appears for the CSV file
BX.4.10.2   The filename reflects the name you gave the export and the date requested
BX.4.11 Save the CSV to your computer    
BX.4.12 Open the CSV in a spreadsheet program    
BX.4.12.1   The CSV contains the data that was displayed in the search results
BX.4.12.2   The CSV is the "admin" CSV, not the "public" CSV. You can tell this because the final 3 columns include account information
