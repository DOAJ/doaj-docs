2. Withdraw from DOAJ

Test context
  • role: admin
Action Expected Result
Go to the admin applications search page at /admin/admin_site_search
Restrict search journals to smaller number (eg. choose only "Subject" -> "Technology" and "Country"-> "Spain")
From the "Journals vs. Articles" assets on the left choose Journals and from "in DOAJ?" choose "True" (note how many journals are not in DOAJ for future reference)
  • Correct restricted search results appear, note the summary number of journals found for future reference
From Bulk Edit Records "Select action..." pull down choose "withdraw from DOAJ" and click "Submit" button
  • Pop up appears: "This operation will affect #nr journals" (#nr is number of journals, the same as number of journals in search result from 2.3.1)
  • Options "Cancel" and "OK" is available
Click "Cancel" on pop up
  • The operation is cancelled
Click "Submit" button again
Click "OK" on the pop-up
  • Message banner appears on the top of the page that includes:
  • Numer of journals queued for edit
  • Link to the background job interface
  • A link to dismiss the message
Click on the link provided to see the background job that was created
  • The set_in_doaj background job is created
Click "More Information"
  • set_in_doaj__journal_ids parameter contains correct number of journals' ids and set_in_doaj__in_doaj -- true parameter is set
Close the background jobs window
Click dismiss on the message banner
  • Banner disappears
