Test context
  • role: Publisher
ID Action Expected Result

BK.3.8 Click the button 'Yes, delete it' in the pop-up box    
BK.3.8.1   You are shown a message whch confirms that your draft has been deleted and a link to the Publisher's dashboard
BK.3.9 Click the link to the dashboard    
BK.3.9.1   You are taken back to the dashboard and your draft has been deleted.
