Language Translation: Change language application Form

1. Change language when logged in

Test context
  • role: publisher
ID Action Expected Result  
BJ.1.1 Make sure you logged in to the application    
BJ.1.2 Go to the `Apply` -> `Application Form`    
BJ.1.3 Click the dropdown which shows `English`, `Français`. the dropdown box is located on the right side above the `Application Progress` menu    
BJ.1.4 Select `Français` if you are in English language or `English` if you are in French language.    
BJ.1.4.1   The text on the application form changes as per the selection. If you select `Français`, the text should change to French language

2. Change language when NOT logged in

Test context
  • role: publisher
ID Action Expected Result  
BJ.2.1 Make sure you are logged out of the application    
BJ.2.2 Go to the `Apply` -> `Application Form`    
BJ.2.2.1   You will be redirected to login page
BJ.2.3 Click the dropdown which shows `English`, `Français`. the dropdown box is located on the right side above the `Application Progress` menu    
BJ.2.4 Select `Français` if you are in English language.    
BJ.2.4.1   The text on the application form changes to French language
BJ.2.5 Enter the credentials and login    
BJ.2.5.1   You are logged in and redirected to Application Form
BJ.2.5.2   The text on the application form is in French language
