1. Editor Todo List

Test context
  • role: editor
  • testdrive: todo_editor_associate
Setup:

Use the todo_editor_associate testdrive to setup for this test, of follow the next steps

You should set up a user account which has both the editor and associate editor roles, and is the editor of at least one editorial group

The user account should be assigned at least 4 applications which meet the following criteria: one that was created over 6 weeks ago, one that has not been modified for 3 weeks, one which has recently been assigned to the user and is in the pending state, and one recent application

The user account should be assigned another 4 applications which meet the following criteria: one that is in the completed state and assigned to your editorial group, one that is assigned to your editorial group in the pending state but with no associate editor assigned, one that is in your editorial group which was created less than 8 weeks ago but which hasn't been updated for 6 weeks, and one that is in your editorial group which was created more than 8 weeks ago
