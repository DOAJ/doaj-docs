Test context

You may change your search text if no results are found, to demonstrate that each field works correctly BS.1.8 Change the page size, using the page-size pull-down BS.1.9 Click the "Next" button to go to the next page of results BS.1.10 Click "Next" again BS.1.11 Click "Prev" BS.1.12 Click "First" BS.1.13 Click the Red X button to clear all filters

2. Test Admin Application Search Display

Test context
  • role: admin
BS.2.1 Go to the DOAJ admin application search at /admin/applications BS.2.2 Open the "Application Status" facet and select "Accepted" BS.2.3 Click on "View request" for one of the results BS.2.4 Click "Back" BS.2.5 Clear the "Accepted" filter and select "Rejected" BS.2.6 Click "Back" BS.2.7 Clear the "Accepted" filter and select any status other than "Accepted" or "Rejected" BS.2.8 Click on "View request" for one of the results
ID Action Expected Result  
BS.2.1 Go to the DOAJ admin application search at /admin/applications    
BS.2.2 Open the "Application Status" facet and select "Accepted"    
BS.2.2.1   Every result includes the text "This update request has been accepted, and your journal in DOAJ updated (view request)." on the bottom right
BS.2.3 Click on "View request" for one of the results    
BS.2.3.1   You are taken to the edit form for the application but it is read only (there is a warning displayed at the top of the page "Please note this form is read-only, you cannot make changes here."
BS.2.4 Click "Back"    
BS.2.5 Clear the "Accepted" filter and select "Rejected"    
BS.2.5.1   Every result includes the text "This update request has been rejected, and your journal in DOAJ updated (view request)." on the bottom right
BS.2.6 Click "Back"    
BS.2.7 Clear the "Accepted" filter and select any status other than "Accepted" or "Rejected"    
BS.2.7.1   Every result includes the text "This update request is currently being reviewed by an Editor (view request)." on the bottom right
BS.2.8 Click on "View request" for one of the results    
BS.2.8.1   You are taken to the edit form for the application but it is read only (there is a warning displayed at the top of the page "Please note this form is read-only, you cannot make changes here."
