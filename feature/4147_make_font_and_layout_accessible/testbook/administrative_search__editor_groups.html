ID Action Expected Result

E.1.5.1   If there are search results, the first page is displayed
E.1.5.2   If there are no search results, suitable "No results to display" text is displayed
E.1.6 Clear your filter and change the page size, using the page-size pull-down    
E.1.6.1   The size of the page of results displays increases appropriately
E.1.7 Click the "Next" button to go to the next page of results    
E.1.7.1   The next page of results is displayed
E.1.7.2   The "First" and "Prev" buttons are activated
E.1.8 Click "Prev"    
E.1.8.1   You are navigated one page forward and then one page back in the results
E.1.9 Click "First"    
E.1.9.1   You are taken to the first page of the results
E.1.10 Click on an editor    
E.1.10.1   You are taken to the user's account page
E.1.11 Click back on the browser    
E.1.12 Click on an associate editor    
E.1.12.1   You are taken to the user's account page
E.1.13 Click back on the browser    
E.1.14 Click "Delete this group" under any chosen group    
E.1.14.1   Confirmation window pops up
E.1.15 Click "Cancel"    
E.1.15.1   The confirmation pop up closes
E.1.15.2   The group is not deleted
E.1.16 Click "Delete this group" again, and click "OK"    
E.1.16.1   A pop up confirms the operation has completed successfully
E.1.16.2   The group disappears from the search results
E.1.17 Click "Edit this group" next to any chosen group    
E.1.17.1   A new browser window opens at the page to edit the group
E.1.18 Close the new browser window    
E.1.19 Click the black X button to clear all filters    
E.1.19.1   The interface returns to the initial state
