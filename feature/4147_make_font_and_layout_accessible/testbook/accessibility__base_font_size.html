1. Test Changing Base Font Size Through Browser Settings

Test context
  • role: any
ID Action Expected Result  
A.1.1 Open any DOAJ page in Google Chrome    
A.1.2 Open Chrome Settings from the menu (⋮ > Settings)    
A.1.3 Navigate to Appearance > Font size    
A.1.4 Change the font size to "Very small"    
A.1.5 Refresh the page    
A.1.5.1   Text on the website becomes noticeably smaller
A.1.5.2   Page layout remains usable and readable
A.1.6 Change the font size to "Very large"    
A.1.7 Refresh the page    
A.1.7.1   Text on the website becomes noticeably larger
A.1.7.2   Page layout adjusts appropriately (no overlapping or cut-off text)
A.1.8 Reset the font size back to "Medium (recommended)"    
A.1.8.1   Website text returns to default size
A.1.9 Repeat this test for other subpages    
A.1.10 Open any DOAJ page in Mozilla Firefox    
A.1.11 Open Firefox Settings from the menu (☰ > Settings)    
A.1.12 Navigate to General > Language and Appearance > Fonts and Colors > Advanced    
A.1.13 Increase the Minimum font size to a larger value (e.g., 24px)    
A.1.14 Refresh the page    
A.1.14.1   Text on the website becomes larger while respecting minimum font size
A.1.14.2   Page remains usable and accessible without broken layouts
A.1.15 Reset the minimum font size to default (e.g., "None" or 0)    
A.1.16 Refresh the page    
A.1.16.1   Website text returns to default size
A.1.17 Repeat this test for other subpages
