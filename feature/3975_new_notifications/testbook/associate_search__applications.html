1. Test Associate Application Search

Test context
  • role: editor
ID Action Expected Result  
O.1.1 Go to the DOAJ associate editor's application search at /editor/your_applications    
O.1.1.1   All available records are found, and the first page of results is shown
O.1.2 Open and Collapse all the facets on the left    
O.1.2.1   The facet opens and closes as expected
O.1.2.2   Each facet contains a maximum of 10 values relevant to that field
O.1.3 Open one of the facets of your choice    
O.1.4 Select one of the values for that facet    
O.1.4.1   All the results for the selected constraints are found, and the first page is shown
O.1.4.2   The filter applied is shown in the facets on the left and above the search results
O.1.5 Go through each sort option in the "sort by" pull-down    
O.1.6 In each case, click the "sort by" button to reverse the direction of sort    
O.1.6.1   The current search results are ordered correctly according to the selected sort field and the selected sort direction
O.1.7 Enter a search term in the search box    
O.1.7.1   All the results for the selected constraints are found, and the first page is shown
O.1.7.2   Your search text remains in the search box
O.1.8 Go through each field constraint under the "search all" pull-down. You may change your search text if no results are found, to demonstrate that each field works correctly    
O.1.8.1   If there are search results, the first page is displayed
O.1.8.2   If there are no search results, suitable "no results match" text is displayed
O.1.9 Change the page size, using the page-size pull-down    
O.1.9.1   The size of the page of results displays increases appropriately
O.1.10 Click the "Next" button to go to the next page of results    
O.1.10.1   The next page of results is displayed
O.1.10.2   The "First" and "Prev" buttons are activated
O.1.11 Click "Next" again    
O.1.12 Click "Prev"    
O.1.12.1   You are navigated one page forward and then one page back in the results
O.1.13 Click "First"    
O.1.13.1   You are taken to the first page of the results
O.1.14 Click the black X button to clear all filters    
O.1.14.1   The interface returns to the initial state
O.1.15 Choose either Accepted or Rejected value in Application Status facet on the left    
O.1.15.1   All result applications have "View finished application" or "View finished update" link
O.1.16 Click on "View finished application/update" link    
O.1.16.1   editor/application/<id> page is opened in the new tab
O.1.17 Click "Unlock and Close" and close the tab    
O.1.18 Click the black X button to clear all filters    
O.1.19 Choose a value in Application Status facet on the left which is not "Accepted" or "Rejected"    
O.1.19.1   All result applications have "Review application" or "Review update" link on the lower right corner
O.1.20 Click on "Review application" link    
O.1.20.1   editor/application/<id> page is opened in the new tab
