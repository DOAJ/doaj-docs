1. Publisher without preservation role

Test context
  • role: publisher
ID Action Expected Result  
N.1.1 Go to publisher area /publisher    
N.1.1.1   Cannot see 'Preservation' tab

2. Publisher with preservation role

Test context
  • role: publisher
ID Action Expected Result  
N.2.1 Go to publisher area /publisher    
N.2.1.1   'Preservation' tab is visible

3. Upload correctly structured file and it is processed correctly

Test context
  • role: publisher
ID Action Expected Result  
N.3.1 Ensure that the archive package has been created with correct structure    
N.3.2 Go to preservation area /publisher/preservation    
N.3.3 Select 'Browse' and select the paaackage zip file    
N.3.4 click 'Upload'    
N.3.4.1   A flash message appears at the top of the screen: File uploaded and waiting to be processed.
N.3.4.2   Upload status is shown in 'History of uploads' with status processing
N.3.5 wait a short amount of time for the job to process, then reload the page (do not re-submit the form data). If the job remains in "pending", reload the page until the status changes.    
N.3.5.1   Upload status in 'History of uploads' changes to 'successfully processed' and the 'Notes' shows as 'uploaded to archive server'

N.5.3 Select 'Browse' and select the paaackage zip file    
N.5.4 click 'Upload'    
N.5.4.1   A flash message appears at the top of the screen: File uploaded and waiting to be processed.
N.5.4.2   Upload status is shown in 'History of uploads' with status pending
N.5.5 wait a short amount of time for the job to process, then reload the page (do not re-submit the form data). If the job remains in "pending", reload the page until the status changes.    
N.5.5.1   upload status in 'History of uploads' to 'processing failed' if all the articles are not owned by the user. If some of the articles are not owned and remaining artcles are owned by the user, the status shows as 'partially succes' if other articles are processed successfully.
N.5.5.2   if the status is 'partially success', show details link will be displayed. When clicked on the link details of how many articles successful and how many not successful will be displayed
