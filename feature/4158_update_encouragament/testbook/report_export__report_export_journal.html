2. Date range histograms

Test context
  • role: Admin
ID Action Expected Result  
CA.2.1 Go to the journal search page in the admin area

Application Link: http://testdoaj.cottagelabs.com//admin/ 		   
CA.2.1.1   At the bottom of the list of facets is a "Reporting Tools" section, including 3 options for working with dates "limit by date range", "date added histogram", "last update histogram" and "date rejected histogram"
CA.2.2 Click on "Date Added Histogram"    
CA.2.2.1   A histogram of the number of journals created per day for the entire life of the DOAJ is displayed, grouped by year
CA.2.3 Click on one of the years    
CA.2.3.1   The search updates, and only records created in that year are displayed
CA.2.3.2   The search filter list at the top of the results indicates the created date range (it may indicate the start and end months of the year)
CA.2.4 Go back to the Date Added Histogram facet, and change the "Granularity" from "Year" to "Month"    
CA.2.4.1   The histogram updates to show the number of journals created per month for the selected year
CA.2.5 Click on one of the months    
CA.2.5.1   The search updates, and only records created in that month are displayed
CA.2.5.2   The search filter list at the top of the results indicates the selected month
CA.2.6 Go back to the Date Added Histogram facet. Above it is the "Limit by Date Range" facet, open this by clicking on it    
CA.2.6.1   The "Limit by Date Range" facet opens, showing a date range picker
CA.2.6.2   Beneath the date range picker the "Created Date" month you selected is also shown
CA.2.7 Remove the Created Date limit by clicking the "X" either in the "Limit by Date Range" section or at the top of the search results    
CA.2.7.1   The search results update to show all records
CA.2.8 Repeat the above steps for the "Last Update Histogram"    

3. Date range limiting

Test context
  • role: Admin
ID Action Expected Result  
CA.3.1 Go to the journal search page in the admin area

Application Link: http://testdoaj.cottagelabs.com//admin/ 		   
CA.3.1.1   At the bottom of the list of facets is a "Reporting Tools" section, including 3 options for working with dates "limit by date range", "date added histogram", "last update histogram" and "date rejected histogram"
CA.3.2 Click on "Limit by Date Range"    
CA.3.2.1   A date range picker is displayed
CA.3.3 Open the "Type" pull-down menu    
CA.3.3.1   The options are "Created Date", "Last Updated", "Last Full Review", "Last Reinstated", "Last Withdrawn", "Last Owner Transfer" and "Date Applied"
CA.3.4 If not already selected, select "Created Date"    
CA.3.4.1   The From and To dates show the month and year of the earliest and latest created dates in the search results, respectively
CA.3.5 Change the "Type" to "Last Updated"    
CA.3.5.1   The From and To dates show the month and year of the earliest and latest last updated dates in the search results, respectively
CA.3.5.2   Note that January 1970 is the date used for records with no last updated date
CA.3.6 Select a new Year value for the Last Updated date range and click "Apply"    
CA.3.6.1   The search results update to show only records that have been updated in the selected range
CA.3.6.2   The search filter list at the top of the results indicates the last updated date range, including the month and year
CA.3.6.3   The "Limit by Date Range" facet also shows the search filter for the Last Updated date range
CA.3.7 Try changing the month and year in both From and To fields for Last Updated in the same way    
CA.3.7.1   The search results and displayed filters reflect the new date ranges
CA.3.8 Choose a "To" date that is earlier than the "From" date    
CA.3.8.1   The "Apply" button is disabled
CA.3.9 Change the "Type" back to "Created Date"    
CA.3.10 Select a new range for the Created date range and click "Apply"    
CA.3.10.1   The "Last Updated" date filter is still active
CA.3.10.2   The search results update to show only records that have been created in the selected range (and were updated in the selected range)
CA.3.10.3   The search filter list at the top of the results indicates the created date range and the last updated range, including the month and year
CA.3.10.4   The "Limit by Date Range" facet also shows the search filter for the Created date range and the last updated range
CA.3.11 Remove both the Created Date and Last Updated date ranges by clicking the "X" in the "Limit by Date Range" section    
CA.3.11.1   The search results update to show all records

4. Export Search as CSV

Test context
  • role: Admin
ID Action Expected Result  
CA.4.1 Go to the journal search page in the admin area

Application Link: http://testdoaj.cottagelabs.com//admin/ 		   
CA.4.1.1   At the bottom of the list of facets is a "Reporting Tools" section, including 3 options for working with dates "limit by date range", "created date histogram" and "last update histogram"
CA.4.2 Open the "Limit by Date Range" facet and set a date range which gives you a subset of the data    
CA.4.3 Apply any other filters from the facets to further refine the search results    
CA.4.4 Click on "Export Data as CSV"    
CA.4.4.1   An export section expands which includes a section to export the search results
CA.4.5 Enter a name for the export in the input box and click "Generate"    
CA.4.5.1   a pop-up appears with a message that the export is being generated
CA.4.6 Close the popup    
CA.4.7 Go to the background jobs page in the admin area

Application Link: http://testdoaj.cottagelabs.com//admin/background_jobs 		   
CA.4.7.1   The export job is listed with the name "admin_reports by [your username]" (it may have a status of queued or processing, or even complete)
CA.4.8 Wait for the job to complete (this can take up to 10 minutes). You will need to refresh the page until it is in the "complete" status.    
CA.4.9 Open your "Notifications" menu at the top of the page    
CA.4.9.1   A notification appears that the report is ready for download
CA.4.9.2   A second notification saying a background job is finished will also be present
CA.4.10 Click on the notification that the report is ready for download    
CA.4.10.1   A download box appears for the CSV file
CA.4.10.2   The filename reflects the name you gave the export and the date requested
CA.4.11 Save the CSV to your computer    
CA.4.12 Open the CSV in a spreadsheet program    
CA.4.12.1   The CSV contains the data that was displayed in the search results
CA.4.12.2   The CSV is the "admin" CSV, not the "public" CSV. You can tell this because the final 6 columns include account information and key dates
