ID Action Expected Result

W.2.2 Login as `aseditor1`    
W.2.3 Go to Journal page at `/editor/journal/1234`    
W.2.4 Click "ADD" button in "Notes" area    
W.2.4.1   New note UI is displayed for you to fill in
W.2.5 Fill in the note    
W.2.6 Click "Save" button

W.2.6.2   you username and user id will be showed on that note
W.2.6.3   the date and time you created the note will be shown

W.2.8 Click "Remove" underneath the note    
W.2.8.1   The note is deleted
